安装与运行方法

以下说明假设你使用的是类 Unix 系统环境。

如果你的系统尚未安装 Python3，请前往 https://www.python.org/downloads/
 下载并安装。同时请按照 https://pip.pypa.io/en/latest/installation/#ensurepip
 中的说明，确保你已安装最新版的 pip。

进入 Code 目录：

cd Code


安装所有依赖：

pip install -r requirements.txt


运行 main.py：

python3 main.py


注意：你可能需要在终端中点击提示，以允许程序访问你电脑的摄像头。
